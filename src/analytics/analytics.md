# Analytics

Purpose: Analytics contains tools and utilities for operational and strategic analysis of grid data, KPI computation, and reporting. It exposes helpers to compute reliability indices, carbon accounting, congestion metrics, and dashboards used by higher-level applications.

Responsibilities:

- Grid KPI computation (SAIDI/SAIFI/ENS, LCOE, NPV)
- Carbon accounting and intensity forecasting
- Time-series KPI trending and anomaly scoring
- Operational dashboards and reports

See the top-level README for examples and usage patterns.

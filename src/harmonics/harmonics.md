# Harmonics

Purpose: Harmonics contains analysis tools for waveform spectral analysis, THD computation, and passive filter design.

Responsibilities:

- THD / THVD calculation using OxiFFT
- Goertzel algorithm for targeted orders
- IEC/IEEE compliance checks (IEC 61000, IEEE 519)
- Filter design helpers (single-tuned, C-type)

Used by power quality and mitigation workflows.
